# Olist E-Commerce Data Analytics (Python)

## Project Overview
This project focuses on analyzing the Olist e-commerce dataset using Python to derive
business insights related to orders, revenue, and performance trends.

The project follows a structured data analytics workflow including data cleaning,
exploratory data analysis, time-series trend analysis, KPI tracking, and insight reporting.

## Dataset
- Source: Olist E-commerce Public Dataset
- Size: ~50,000+ rows
- Tools Used: Python (Pandas, NumPy, Matplotlib, Seaborn)

## Objectives
- Clean and transform raw e-commerce data
- Perform exploratory data analysis (EDA)
- Analyze time-based trends in orders and revenue
- Identify key performance indicators (KPIs)
- Generate actionable business insights

## Key Steps Performed
- Data cleaning and preprocessing using Pandas
- Handling missing values and inconsistent formats
- Exploratory Data Analysis (EDA) to identify patterns and anomalies
- Time-series analysis to track order and revenue trends
- Visualization of insights using Python charts
- Structured reporting and documentation

## Key Insights
- Identified seasonal patterns in customer orders
- Highlighted revenue contribution trends over time
- Detected anomalies impacting business performance
